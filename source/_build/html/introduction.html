Инструкция для Администратора

  • Настройка подачи заявок для возрастной группы
  • Настройка нового Менеджера первой линии (М1)
  • Настройка нового Методиста индивидуального мастер класса (МИМК)
  • Настройка расписания работы МИМК
  • Настройка нового Руководителя

Инструкция для Менеджера (М1)

  • Получение Заявки для проверки
  • Изменение контактной информации в Заявке
  • Подтверждение Заявки (подтверждение проведения ИМК)
  • Отмена Заявки
  • Возврат Заявки в очередь для повторной проверки
  • Просмотр назначенных Заявок

Инструкция для Методиста (МИМК)

  • Настройка расписания работы МИМК
  • Получение Заявки для проведения ИМК
  • Отмена Заявки (отмена проведения ИМК)
  • Завершение выполнения Заявки (отмена завершения ИМК)
  • Возврат Заявки в очередь для перераспределения
  • Просмотр Заявок назначенных для проведения ИМК

Инструкция для Руководителя

  • Просмотр Заявок возрастной группы
  • Просмотр истории исполнения Заявки
  • Настройка расписания работы МИМК
  • Построение отчета “Утилизация менеджеров”
  • Построение отчета “Рабочее время Менеджеров”

Инструкция для Клиента

  • Подача заявки на Индивидуальный мастер класс
  • Отмена заявки на Индивидуальный мастер класс
