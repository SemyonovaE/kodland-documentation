Настройка расписания работы МИМК и М1
-------------------------------------

Чтобы настроить **расписание работы**:

1. Нажмите на поле **Расписание**.

.. figure:: _static/mimk/mimk9.png
    :scale: 42 %
    :alt: alternate text
    :align: center

2. Нажмите на текст **Добавить промежуток времени**.

.. figure:: _static/mimk/timetable/mimk21.png
    :scale: 42 %
    :alt: alternate text
    :align: center

3. Нажмите на **поле с датой**.

.. figure:: _static/mimk/timetable/mimk22.png
    :scale: 42 %
    :alt: alternate text
    :align: center

4. Установите дату начала и окончания периода, затем нажмите **Установить**.

.. hint:: Вы можете установить любой интервал времени, но для настройки **расписания рабочего дня** необходимо **выбрать равные значения**.

.. figure:: _static/mimk/timetable/mimk23.png
    :scale: 42 %
    :alt: alternate text
    :align: center

5. Нажмите на **поле со временем**.

.. figure:: _static/mimk/timetable/mimk24.png
    :scale: 42 %
    :alt: alternate text
    :align: center

6. Установите время начала и окончания рабочего дня, затем нажмите **Установить**.

.. figure:: _static/mimk/timetable/mimk25.png
    :scale: 42 %
    :alt: alternate text
    :align: center

7. Нажмите на поле **Повтор**.

.. figure:: _static/mimk/timetable/mimk26.png
    :scale: 42 %
    :alt: alternate text
    :align: center

8. Нажмите на поле **Частота**. По умолчанию **Не повторяется**.

.. hint:: По умолчанию **Не повторяется**.

.. figure:: _static/mimk/timetable/mimk27.png
    :scale: 42 %
    :alt: alternate text
    :align: center

9. Выберите подходящий вариант из списка.

.. hint:: Если Вы работаете не каждый день, то выберите значение **Каждую неделю**.

.. figure:: _static/mimk/timetable/mimk28.png
    :scale: 42 %
    :alt: alternate text
    :align: center

10. В появившемся окне выберите те дни, когда Вы работаете.

.. figure:: _static/mimk/timetable/mimk29.png
    :scale: 42 %
    :alt: alternate text
    :align: center

11. Нажмите на поле **Повторять до**. Выберите период времени, до которого необходимо повторять данное расписание.

.. figure:: _static/mimk/timetable/mimk30.png
    :scale: 42 %
    :alt: alternate text
    :align: center

12. Установите **год, месяц и дату** окончания периода.

.. figure:: _static/mimk/timetable/mimk31.png
    :scale: 42 %
    :alt: alternate text
    :align: center

13. Нажмите |галка|, чтобы сохранить изменения.

    .. |галка| image:: _static/galka.png
        :scale: 42 %

.. figure:: _static/mimk/timetable/mimk32.png
    :scale: 42 %
    :alt: alternate text
    :align: center

14. При необходимости введите **описание** и нажмите |галка|. Ваше расписание **сохранено**, теперь Вы можете продолжить редактирование **Ресурса**.

.. figure:: _static/mimk/timetable/mimk33.png
    :scale: 42 %
    :alt: alternate text
    :align: center
